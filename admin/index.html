Comment ça fonctionne

Les modifications sont committées directement sur le dépôt GitHub via l'API REST.

GitHub Pages redéploie automatiquement en 1–2 minutes après chaque commit.

Les livres et audios utilisent des liens Google Drive (partage public requis).

Les vidéos utilisent les IDs YouTube (ex: dQw4w9WgXcQ).
